A simple stock control developed in PHP using a MySQL and Postgre databases. During this project I learned a lot of concepts of object-oriented in backend and frontend. All the project use a pure OO ES5 with prototypes emulating classes. Jquery was also used to easily access DOM and HTTP requests. I also packaged the application using debian tools which can be installed with APT.

Install with APT Run with Docker Github Project

Technologies used: PHP, MySQL, Postgree, Javascript, JQuery, Bootstrap, Debian package, Docker
