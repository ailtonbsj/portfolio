CREDE OS is an ERP to manage informations about IT Support on educational institutions.

There are several modules like: Employees, Institutions, Internet links, Inventory and Equipments. Employees module we can manage staff data, exports contacts and filter by groups. Institutions module we can manage data like local, geographic coordenations, and others. Inventory and Equipments module we can manage all IT assets creating reports of historic of maintenance, warranty time and others.

With this project I learn frameworks like PHP Laravel and Javascript VueJS and Vuetify.
