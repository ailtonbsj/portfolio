Essa ferramenta facilita o chaveamento rápido das configurações de proxy no ambiente linux. Já está compatível com o Bionic Beaver.

Modifica as configurações de proxy dos seguintes apps:

  • Firefox, Google Chrome e Chromium
  • Arquivo apt.conf e wgetrc
Instalação automatizada via npm
sudo apt update
sudo apt install npm
sudo npm i -g --unsafe proxy-linux

Acesse o projeto no Github.

Linguagem utilizada: Shell script
