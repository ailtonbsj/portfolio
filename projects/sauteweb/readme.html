SauteWeb is an Progressive Web App based on another app created by Mario Rocha in lazarus (FreePascal).

This app manages data and generation of reports about temporary authorization in educational institutions.

With this project I learned how to use Angular 14 and Angular Material UI library. The app can run offline using IndexedDB with Dexie.js library. It is possible to switch the database to use an external API, which was developed using Java and Spring Boot Framework.

Developing the API I used Spring Security to generate authentication and autorization per roles. The authentication was based on Auth0 JWT. All development was using Visual Code IDE.
