SauteWeb is a Progressive Web App based on another app created by Mario Rocha in Lazarus (FreePascal). This app manages data and generation of reports about temporary authorization in educational institutions.

With this project, I learned how to use Angular 14 and Material UI library. Developing the API I used Spring Security to generate authentication and authorization per roles.

The authentication was based on Auth0 JWT. All development was using Visual Code IDE. The app can run offline using IndexedDB with Dexie.js library.

It is possible to switch the database to an external API, which was developed using Java and Spring Boot Framework. A Google login button was implemented to use GSuite e-mail.
