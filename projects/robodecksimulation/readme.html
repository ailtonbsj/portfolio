Este é o simulador do Robodeck desenvolvido usando Python com Pygame. O Robodeck é uma plataforma robótica para uso geral e permite utilizar diversos sensores.

O simulador contém:

  • Modulo python robodeck_pkg (O simulador)
  • Exemplo de arquitetura Planejador-reator
  • Exemplo de uso de funções do modulo
  • Diagrama de classes
