An app to allocate equipments and rooms in a shared Schedule. A framework named Orkidea was developed to improve this project. The system was deployed for a long time on Heroku using Postgre database.

Recently this application was integrated with Google Calendar, where the schedule can be shared and receive notification using the app Google Calendar, native in Android OS.

Demo on desktop Demo on Smartphone

Technologies used: PHP, Javascript, PostgreSQL, Google Calendar API, Node, Docker
