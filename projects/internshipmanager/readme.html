InternshipManager is an app capable of managing student documents for yours internships, avoiding to typing a lot of files and stored in a postgres database. With this project I learn how to use PostgreSQL and pgAdmin, learn Java and Swing Framework using Eclipse IDE.

Recently a packed this project as a study of Docker with Webswing, which can run SwingApps on browser.

Run with Docker Github Project SourceForge (Win)

Technologies used: Java, Swing, PostgreSQL, Webswing, Docker
