CREDE ERP is an authentication server, user and organization management system. Users can log in using crendentials or clicking on Google Login Button. The system provide access to internal systems using a JWT token. The User management has the concepts of roles and permission permitting levels of granularity in access to systems.

In this project I learned how to use the stack Angular/Spring. I also learned how to use Oracle Cloud, how to use TomCat server, deploy a SSL certificate on an VPS, use Google Cloud Platform, deploy frontend on Firebase and others things.

Technologies used: Angular 15, Angular Material, Spring Boot, Spring Security, PostgreSQL, Oracle Cloud, Firebase
